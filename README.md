# 🌐 Urbanblitz - Desarrollo Web Frontend para Marca de Ropa Urbana (En Proceso)

Este repositorio contiene la estructura inicial, el diseño de interfaz y la maquetación estática para **Urbanblitz**, mi futura marca de ropa urbana (streetwear). 

El proyecto se encuentra actualmente en **fase activa de desarrollo y diseño frontend**, sirviendo como base sólida para controlar al 100% la estética y la limpieza del código sin dependencias externas en sus primeras etapas.

---

## 🚀 Estado del Proyecto y Hoja de Ruta (Roadmap)

⚠️ **Nota importante:** El sitio web está en proceso de creación. La estrategia consiste en finalizar primero toda la arquitectura visual y el código base de manera nativa para, posteriormente, dotar a la plataforma de total funcionalidad comercial.

### Próximos pasos en el desarrollo:
* [x] Estructuración del esqueleto semántico (HTML5).
* [x] Diseño estético, identidad de marca, sistema de colores y tipografías.
* [ ] Optimización de la adaptabilidad (Responsive Design) para dispositivos móviles.
* [ ] **Migración a Framework para Producción:** Una vez que la estética y el código frontend estén completamente acabados, la idea principal es exportar todo el proyecto a un framework moderno (como **React** o **Vue.js**). Esto permitirá darle vida útil a la plataforma mediante la integración de lógica dinámica, gestión de stock, carrito de compras y pasarelas de pago automatizadas.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5** - Estructuración semántica del contenido de la tienda y las secciones.
* **CSS3** - Maquetación visual, diseño de interfaz personalizada, estilos y estética de marca.
* **JavaScript** - Lógica Frontend nativa e interacciones dinámicas con el usuario, asentando las bases para la futura migración a entornos reactivos.
